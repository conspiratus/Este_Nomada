# Руководство по управлению кнопками Hero

## Описание

Теперь кнопки в разделе Hero можно настраивать через админку Django. Вы можете:
- Создавать любое количество кнопок
- Настраивать их порядок отображения
- Выбирать визуальный стиль
- Настраивать ссылки (внутренние или внешние)
- Добавлять переводы для каждой кнопки

## Как добавить кнопку

1. Перейдите в админку: `/admin/core/herobutton/add/`

2. Заполните поля:
   - **Порядок отображения** - чем меньше число, тем раньше кнопка отображается (0, 1, 2...)
   - **URL ссылки** - может быть:
     - Относительным: `/ru/stories`, `/es/order`, `/en/account`
     - Абсолютным: `https://t.me/este_nomada`, `https://instagram.com/...`
   - **Стиль кнопки** - выберите один из вариантов:
     - **Primary** - Основная (залитая кнопка, оранжевая)
     - **Secondary** - Вторичная (залитая кнопка, темная)
     - **Outline** - Контурная (прозрачная с белой рамкой)
     - **Ghost** - Призрачная (прозрачная без рамки)
   - **Активна** - показывать ли кнопку на сайте
   - **Открывать в новой вкладке** - добавляет `target="_blank"`

3. Добавьте переводы:
   - В разделе "Переводы" добавьте переводы для каждого языка:
     - **Локаль**: `ru`, `es`, `en`
     - **Текст кнопки**: текст, который будет отображаться на кнопке

4. Сохраните кнопку

## Примеры настройки

### Кнопка "Читать истории"
- Порядок: `0`
- URL: `/ru/stories` (или `/es/stories`, `/en/stories` - будет автоматически подставляться локаль)
- Стиль: `Primary`
- Активна: ✅
- Открывать в новой вкладке: ❌
- Переводы:
  - ru: `Читать истории`
  - es: `Leer historias`
  - en: `Read stories`

### Кнопка "Телеграм-канал"
- Порядок: `1`
- URL: `https://t.me/este_nomada`
- Стиль: `Outline`
- Активна: ✅
- Открывать в новой вкладке: ✅
- Переводы:
  - ru: `Телеграм-канал`
  - es: `Canal de Telegram`
  - en: `Telegram Channel`

## Визуальные стили

### Primary (Основная)
- Залитая оранжевая кнопка
- Белый текст
- Тень и эффект увеличения при наведении
- Используется для главных действий

### Secondary (Вторичная)
- Залитая темная кнопка
- Белый текст
- Тень
- Используется для второстепенных действий

### Outline (Контурная)
- Прозрачная кнопка с белой рамкой
- Белый текст
- Легкое затемнение при наведении
- Используется для внешних ссылок

### Ghost (Призрачная)
- Полностью прозрачная кнопка
- Белый текст
- Легкое затемнение при наведении
- Используется для минималистичного дизайна

## Важные замечания

1. **Порядок отображения**: Кнопки отображаются в порядке возрастания поля "Порядок отображения"
2. **Локализация**: Обязательно добавьте переводы для всех языков (ru, es, en), иначе кнопка может не отображаться корректно
3. **Внутренние ссылки**: Если URL начинается с `/`, система автоматически определит это как внутреннюю ссылку и использует компонент `Link` из Next.js
4. **Внешние ссылки**: Если URL начинается с `http://` или `https://`, используется обычный тег `<a>`
5. **Активность**: Только активные кнопки (`active=True`) отображаются на сайте

## Миграция существующих кнопок

После применения миграции нужно вручную создать кнопки в админке, так как старые хардкодные кнопки больше не работают.
