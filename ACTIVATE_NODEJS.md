# Как активировать Node.js в панели one.com

## Пошаговая инструкция

### Шаг 1: Вход в панель управления

1. Откройте https://www.one.com/admin
2. Войдите в свой аккаунт

### Шаг 2: Найти раздел Node.js

**Вариант A: Через меню "Websites"**
1. В главном меню найдите **"Websites"** или **"Домены"**
2. Выберите домен **estenomada.es**
3. В меню домена найдите **"Node.js"** или **"Applications"**

**Вариант B: Прямой поиск**
1. В верхней панели поиска введите **"Node.js"**
2. Выберите соответствующий раздел

### Шаг 3: Активировать Node.js

1. Нажмите кнопку **"Enable Node.js"** или **"Activate Node.js"**
2. Выберите версию Node.js (рекомендуется **18.x** или **20.x**)
3. Если есть выбор между стабильной и LTS версией — выберите **LTS**

### Шаг 4: Настроить приложение

Заполните следующие поля:

- **Application name** (название приложения): `Este Nomada` или `estenomada`
- **Entry point** (точка входа): `server.js`
- **Start command** (команда запуска): `npm start` или `node server.js`
- **Working directory** (рабочая директория): `/customers/d/9/4/czjey8yl0/webroots/17a5d75c`
- **Port** (порт): `3000` (или оставьте по умолчанию, если one.com автоматически назначает порт)

### Шаг 5: Сохранить и запустить

1. Нажмите **"Save"** или **"Deploy"**
2. Дождитесь активации (обычно 1-2 минуты)

### Шаг 6: Установить зависимости

После активации Node.js, подключитесь по SSH:

```bash
ssh -p 22 czjey8yl0_ssh@ssh.czjey8yl0.service.one
```

Пароль: `Drozdofil12345!`

Затем выполните:

```bash
cd /customers/d/9/4/czjey8yl0/webroots/17a5d75c
npm install --production
```

### Шаг 7: Перезапустить приложение

В панели one.com:
1. Найдите ваше Node.js приложение
2. Нажмите **"Restart"** или **"Reload"**

   - Нужна версия Node.js 18+ или 20+

## Проверка работы

После настройки:

1. Откройте https://estenomada.es
2. Должна открыться главная страница лендинга
3. Если видите ошибку — проверьте логи в панели one.com

## Возможные проблемы

### "Node.js not found"
- Убедитесь, что Node.js активирован в панели
- Проверьте, что указан правильный путь к рабочей директории

### "Cannot find module"
- Выполните `npm install --production` в рабочей директории

### "Port already in use"
- Измените порт в `server.js` или настройках панели
- one.com может автоматически назначать порт — проверьте в панели

### Сайт не открывается
- Проверьте, что домен правильно настроен
- Убедитесь, что приложение запущено (статус "Running" в панели)
- Проверьте логи приложения в панели one.com

```bash
# Проверить процессы Node.js
ps aux | grep node

# Проверить логи (если доступны)
tail -f /var/log/nodejs/estenomada.log
```
